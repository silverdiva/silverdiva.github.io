Topic #10 HTML5 Tags for Media

Video, Audio, and Canvas

Using the HTML5 Audio Tag

Using the *control* attribute for audio playback.

With the *audio* tag we can embed an audio file within our HTML page.
Here is a beautiful piano piece by Chad Crouch called "Ginger", which can be found here.

Using the *audio controls* tag, the embedded audio file has "built-in" controls to play and pause content, fast-forward and reverse it, as well as control the volume."

Without adding *control* to the *audio* tag, any embedded content plays automatically, which can be useful for specific situations.


Using JavaScript and "onclick" events to control playback.

We still have an embedded audio file, but instead of using the *control* attribute within the *audio* tag, we can use buttons with "onclick" events to call JavaScript functions to control playback.

Click the buttons to play or pause the audio.




Using the HTML5 Video Tag

Using the *control* attribute for video playback.

Just like with embedding an audio file with automatic controls, we can embed a video file within our HTML page using the same *control* attribute.
Here is a beautiful time-lapse from the Pexels site by Felix Mittermeier, which can be found here.

Using JavaScript and "onclick" events to control playback.

Just like the audio file above, we still have our beautiful embedded video file, but instead of using the *control* attribute within the *video* tag, we can use buttons with "onclick" events to call JavaScript functions to control playback.

Click the buttons to play or pause the video.



Using the HTML5 Canvas Tag

Do cool things with the *canvas* tag.

With the *canvas* tag we can make a 'virtual canvas' within an HTML page.



Draw Lines

Use JavaScript functions and specialized methods to "draw within the canvas" with onclick events that call functions to render stroke paths text, images, or animations.
This is a very simple canvas that utilizes the "moveTo(), lineTo(), closePath(), and stoke()" methods to "draw" a simple set of intersecting lines.



Draw Styled Lines

Style the stoke paths using the "strokestype" property and methods such as "createLinearGradient()" and "addColorStop()" to give your coloring some pizzaz.
