JSON Parse, Stringify | Using XMLHTTPRequest to Consume a JSON Web Service

Using AJAX XMLHTTPRequest, I am connecting to the OpenWeatherMap.org API web service to grab a JSON object for current weather conditions based on a user-input search string.

Using stringify, the returned JSON object is parsed into readable format and displayed as the current temperature and humidity values.

A few things are going on in the background:

  • User input (string value in the search box) triggers the "findWeatherDetails()" function.
  • Which pulls weather data from the OpenWeatherMap.org web service API as a raw JSON object via the "theResponse(response)" function.
  • The raw JSON object (e.g. "jsonObject.name/ .temp/ .humidity" ) is parsed into a Javascript object via the "JSON.parse(response)" method.
  • The parsed JS values are attached to ID's within the HTML referenced by the "xx.innerHTML" property, and called using the "document.getElementById()" method to display these nicely formatted values:
    • cityname (document.getElementById("icon")
    • weather-type icon (document.getElementById("icon")
    • temperature (document.getElementById("temp")
    • humidity document.getElementById("humidity-div")
  • The " JSON.stringify(response)" method displays the JS object as a string.
