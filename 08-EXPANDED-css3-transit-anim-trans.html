Topic #08: CSS3 Transitions, Animations, Transformations Triggered by JS

Designing, Defining, and Triggering CSS3 Transitions without Custom Libraries (From Scratch)

We can use JavaScript to animate the visibility of things such as titles, paragraph text, and lists.

We can then use JavaScript to control the rate or speed at which those things are animated, which is called a transition.


Visibility Transition

In this example, we use JavaScript to "toggle" the 'height' property to control the rate of the 'show/hide visibility' of text.

The current transition rate to show/hide the text is set at 1200 milliseconds using the CSS3 'transition' property with the associated value of 'height 1200ms ease-in-out;'.

If we increase the time to say 2000 milliseconds, then the show/hide transition would be slower.
If we decrease the time to say 500 milliseconds, then the show/hide transition would be faster.
